# IMDb Movie Ratings Analysis

## Project Overview
This project analyzes IMDb movie ratings using Python and Power BI.

The project includes:
- Data Cleaning
- Summary Statistics
- Data Visualization
- Genre Analysis
- Interactive Dashboard

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Power BI

## Dataset
IMDb Top 1000 Movies Dataset

## Features
- Rating Distribution
- Top Rated Movies
- Genre Analysis
- Power BI Dashboard
- Interactive Filters

## Conclusion
The analysis revealed that most movies are rated between 7 and 8, with Drama and Crime genres dominating the top-rated section.
